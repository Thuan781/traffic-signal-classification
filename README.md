# 🚦 Traffic Signal Classification

## 📌 Overview
This project classifies traffic signals (Red, Yellow, Green) from images using image processing techniques in Python. It is built as an academic project for Data Science & AI.

## 🎯 Objective
- Detect and classify traffic signal images  
- Apply image preprocessing techniques  
- Build a simple image classification system  

## 🧠 Tech Stack
- Python  
- OpenCV  
- NumPy  
- Imutils  
- TensorFlow / Keras (if used)  
- Scikit-learn (if used)  

## ⚙️ How It Works
1. Load input image  
2. Resize and preprocess image  
3. Extract features or apply model  
4. Classify traffic signal  
5. Display output result

## 🚀 Installation
pip install -r requirements.txt  

## ▶️ How to Run
python traffic_signal.py  

## 📊 Output
Input Image → Output: RED / GREEN / YELLOW
